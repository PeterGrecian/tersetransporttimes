# NaPTAN Stop IDs

Bus stops used in the TerseTransportTimes app.

## Parklands

- **490010781S** = Inbound (towards Kingston Hospital via Surbiton)
- **490010781N** = Outbound (towards Hook via Tolworth)

## Surbiton Station

- **490015165A** = Stop NC, Inbound (towards Kingston)
- **490015165B** = Stop NK, Outbound (towards Hook)

## References

NaPTAN (National Public Transport Access Nodes) IDs are UK national identifiers for bus stops and transport access points. These IDs are used with TfL (Transport for London) APIs to fetch real-time bus arrival data.
